# WeatherDiaryApp

## Что это?

WeatherDiaryApp - проект, созданный в рамках хакатона от SimbirSoft летом 2020 года

## Зачем это?

Многие школы требуют ведения дневника погоды для школьников. Не все любят это делать. Почему бы не автоматизироватьь это?

Это и послужило главным стимулом к созданию проекта. С помощью нашего продукта вы можете "подписаться" на город - получать готовый к работе дневник погоды.

## Стек технологий

Проект построен с помощью C# на платформе .NET5. Главный фреймворк ASP.NET Core. База данных - PostgreSQL. Чтобы коннектиться к БД - EF Core.
